## Installation of Common Python Libraries Using pip

| Command                     | One-Line Explanation                                                 |
| --------------------------- | -------------------------------------------------------------------- |
| `pip install openai`        | Installs the OpenAI Python library for accessing AI models via API.  |
| `pip install python-dotenv` | Installs a package to load environment variables from a `.env` file. |
| `pip install pandas`        | Installs a library for data analysis and manipulation.               |
| `pip install matplotlib`    | Installs a library for creating charts and visualizations.           |
